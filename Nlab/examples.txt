BEGIN {
   SET $I := 5 ;
   PRINT $I
} 

sets the variable I to have the value 5, and prints it to the screen

Output:
5



You can create an array full of ones and add 2 to each cell of the array:
BEGIN {
   ONES 6 5 $A
   SET $A := $A 2 B - ADD ;
   PRINT " ARRAY :"
   PRINT $A
}

Output:
ARRAY:
3 3 3 3 3
3 3 3 3 3
3 3 3 3 3
3 3 3 3 3
3 3 3 3 3
3 3 3 3 3
